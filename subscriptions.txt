# 每行一个 UP 主，格式：  名字 <空格> 主页/动态链接（或直接写 UID）
#   - 以 # 开头的行是注释，空行忽略
#   - “名字”只是给你看的备注，可随意改；脚本以链接里的 UID 为准
#   - 增删 UP 主：直接改这个文件保存即可
#     （在 GitHub 上改完点提交，下一次运行就会自动生效）
#   - 懒得改文件？在仓库 Settings → Secrets and variables → Actions → Variables
#     建一个 BILI_SUBS（多行，同样的“名字 链接”格式），设了它就以它为准、本文件被忽略，
#     fork 后设一次即可，不用动代码。
